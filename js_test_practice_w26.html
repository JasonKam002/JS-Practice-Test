Question 2 — Temperature Converter   [10 marks]

The user enters a temperature value and selects a conversion direction using radio buttons (°C to °F, or °F to °C). Clicking a button displays the converted result in a result box. The background colour of the result area should turn blue if the result is below 0°C (or 32°F), and red if the result is above 100°C (or 212°F), and white otherwise.

Hint: C to F formula: (C × 9/5) + 32. F to C formula: (F − 32) × 5/9. Access radio buttons with document.querySelector('input[name="direction"]:checked').

Question 6 — Show / Hide Info Panel   [12 marks]

There is a styled div below that contains some text ("Here is your secret info!"). It starts hidden (display: none). A Toggle button controls it:

  • First click: the div becomes visible and the button text changes to "Hide Info". The div background changes to lightblue.
  • Second click: the div is hidden again and the button text changes back to "Show Info".
  • This should continue to alternate every click.

A separate Change Colour button (always visible) cycles the "Secret Info" div background through three colours. Changing once each time it is clicked: lightblue → lightyellow → lightgreen → lightblue … (the div must be visible to see the effect).

Hint: Use a variable (e.g. let isVisible = false;) to track state. Use style.display = "block" or "none". Use textContent to change button text. For colour cycling, try an array and track an index.

Question 7 — Click Box Listener   [12 marks]

There is a box below that says Click Me!. The user will use an Add Listener button to attach a click event to that box. While the listener is active, each click on the box should:

  • Increase a click counter by 1
  • Display the current number of clicks in a result div
  • Cycle the box background through three colours

A Remove Listener button should detach the click event so the box no longer reacts when clicked. A Reset button should set the counter back to 0, return the box to its starting colour, and show a reset message in the result div.

Hint: Use a named function for the click handler so it can be passed to both addEventListener() and removeEventListener(). Use a colour array and an index to cycle the box background.
